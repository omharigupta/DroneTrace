Accepted File Formats
1
Tampering Dataset Format .csv

Time-series telemetry with case_id, row_idx, and label columns.

Columns: case_id, row_idx, latitude, longitude, altitude, speed, ..., label
2
DJI DatCon Export .csv

DJI flight controller log export with GPS and motor telemetry.

Columns: GPS:Lat, GPS:Long, GPS:heightMSL, Motor:Speed:*, ...
3
NIST Drone Forensics .txt .dat

NIST forensic telemetry standard (lat, lon, groundspeed, altitude).

Columns: lat, lon, altitude, groundspeed, ...
4
Generic CSV .csv

Any CSV with standard column names auto-detected.

Required: latitude, longitude, altitude (speed & motor_rpm optional)

How It Works
1
SHA-256 Hash
Computes cryptographic hash for chain-of-custody integrity
2
Parse & Window
Auto-detects format, creates 10-second sliding windows (50% overlap)
3
Transformer Classification
LoRA-adapted Transformer encoder classifies each window as normal/tampered
4
Reference Validation
Compares flight stats against known-good reference profile from real drone data
5
Combined Verdict
Merges model + reference results → CLEAN / TAMPERED / SUSPICIOUS
